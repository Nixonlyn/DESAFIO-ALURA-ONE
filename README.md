# DESAFIO-ALURA-ONE
Este proyecto analiza datos de ventas para identificar patrones y tendencias en el comportamiento de los productos. A través de análisis exploratorio y visualizaciones, se busca comprender qué categorías generan mayor ingreso y cómo esta información puede apoyar la toma de decisiones basadas en datos.


## Dataset
El conjunto de datos utilizado contiene información básica de ventas de productos, incluyendo categoría, precio y cantidad vendida.

## Objetivo
Realizar un análisis exploratorio de datos (EDA) para identificar patrones de ventas y productos con mayor rendimiento.

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib

## Resultados
El análisis permite identificar las categorías con mayor volumen de ventas y comprender el comportamiento de los productos.
